#  G型底座

**适配型号**：myCobot 280、myPalletizer 260、mechArm 270

![图12](../../../resourse/2-serialproduct/图12.jpg)

### 规格说明：

| 名称           | 平面底座                             |
| -------------- | ------------------------------------ |
| 型号 Model     | myCobot_Fstand_grey                  |
| 颜色           | 灰色、黑色                           |
| 工艺           | ABS压塑                              |
| 尺寸 Size      | 145×145×13                           |
| 重量 Weight    | 60g                                  |
| 固定方式 Fixed | 乐高连接件/螺丝固定                  |
| 使用环境要求   | 常温常压                             |
| 适用设备 Fit   | ER myCobot 280 M5，ER myCobot 280 Pi |



### 使用说明：
**G型底座 - 固定在桌子边沿**
- 1.用G形夹将底座固定在桌子边沿

- 2.用附带的乐高科技件，连接底座和机械臂底部

- 3.确定稳固后方可开始使用
